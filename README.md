## Key Design Decisions

**рҹҺЁ Visual Identity:**
- Dark GitHub-native color palette (`#0d1117`, `#58a6ff`, `#ff6e40`) вҖ” looks stunning in both light and dark mode
- Consistent `for-the-badge` styling across all badges
- ASCII art branding at the footer for memorability

**рҹ“җ Layout Strategy:**
- Side-by-side table for "About" + "Stats" вҖ” immediate visual impact
- 2Г—2 project grid вҖ” scannable, not overwhelming
- Each project has a distinct identity (emoji + color accent)

**рҹ§  Content Strategy:**
- The `whoami` Python class is a conversation starter вҖ” shows personality
- Sky Language gets the code sample вҖ” it's your most unique project
- The ASCII architecture diagram for the Gateway shows systems thinking
- The philosophy section differentiates you from every "I use React" profile

**вҡЎ What makes it stand out:**
- No generic "I'm a passionate developer" intro
- The typing SVG has **bold, confident lines** вҖ” not buzzwords
- Projects are described by what they **do**, not what framework they use
- The callout box at the end is a clear CTA without being desperate
